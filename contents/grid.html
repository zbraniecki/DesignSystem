Grid

When all of your measurements follow the same rules, you automatically get a more consistent UI.

Baseline

The grid uses fixed margins and gutters to help control the proportion and balance of every Firefox components. All components, including iconography, align to an 4px baseline grid

This baseline helps with the vertical alignment and spacing of content.

Workflow tip

You can use Nudg.it to set your keyboard increment by 4px in Sketch.
